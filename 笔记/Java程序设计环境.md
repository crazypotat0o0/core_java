Java术语：  
JDK 编写Java程序的程序员使用的软件  
JRE 运行Java程序的用户使用的软件
SE  用于桌面或者简单的服务器应用的Java平台
EE  用于复杂的服务器应用的Java平台
ME  用于手持设备的Java平台
J2  一个过时的术语，用于描述98年到06年间的Java版本
SDK 一个过时的术语，用于描述98年到06年间的JDK
u   用于发布修改的bug，如JDK7u1
